Installation

Takes under a minute on either platform.

macOS

  1. Download the .dmg and open it.
  2. Drag the app into Applications.
  3. First launch: right-click the app → Open → Open.
  4. Click Allow when asked for microphone access.
One universal app for Apple Silicon (M1–M4) and Intel Macs. The right-click step is only needed once — it tells Gatekeeper you trust this (not yet code-signed) app.

Windows

  1. Download and run the Setup .exe.
  2. If SmartScreen appears: More info → Run anyway.
  3. Pick an install location and finish — the app starts automatically.
Prefer no installation? Use the portable .exe — run it from anywhere, including a USB stick (portable builds don't auto-update).
